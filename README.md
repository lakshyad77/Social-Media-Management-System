# Social Media Management System

This is a console-based Java application that simulates a social media platform. It allows users to create profiles, manage friend requests, share posts, and interact through comments.

## Features

- **User Management**: Support for Regular and Admin user types.
- **Friendship System**: Send and manage friend requests between users.
- **Post System**: Create and share content with other users.
- **Interactions**: Add comments to existing posts.
- **Admin Controls**: Administrative privileges to manage and remove users.
- **Data Validation**: Built-in validation for emails (Gmail support) and usernames.

## Technologies Used

- **Language**: Java
- **Architecture**: MVC (Model-View-Controller) pattern
- **Storage**: In-memory database for efficient data handling.

## How to Run

1. Compile the Java files.
2. Run the Main class in the com.socialmedia.main package.
3. Follow the on-screen menu instructions.
